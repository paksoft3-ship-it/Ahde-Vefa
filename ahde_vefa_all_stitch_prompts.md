# AHDE VEFA İnsani Yardım Derneği — Complete Stitch Prompt Pack

This file contains all screen prompts created for the public website and admin panel.

## Global Design System

- Follow `ahde_vefa_design.md` before designing every screen.
- Turkish UI text only.
- Colors: Primary Green `#2A6A48`, Action Turquoise `#04B6CA`, Dark Green `#123D2B`, Soft Mint `#EEF8F4`, Warm Cream `#FFF8EF`, Donation Gold `#F4B740`, Dark Text `#1F2933`, Muted Text `#667085`, Border `#E5E7EB`.
- Style: modern, trustworthy, clean, emotional but not exploitative, Islamic-sensitive, transparent, mobile-first, donation-focused.
- Never invent legal numbers, bank details, prices, dates, reports, statistics, donor/admin names, phone numbers, email addresses, addresses, or completed activities.
- Use `[Eklenecek]` placeholders where real data is missing.
- Pakistan must always be shown as `Hazırlık Aşamasında` unless real verified completed activity is provided.
- Avoid poverty-exploitation imagery. Use dignity-preserving humanitarian visuals.
- Avoid graphic animal imagery for Kurban.

## Standard Public Header
Navigation: Anasayfa, Bağış Yap, Kurban, Ramazan, Faaliyetler, Raporlar, Hakkımızda, İletişim. Right buttons: Bilgi Al, Bağış Yap.

## Standard Admin Layout
Sidebar menu: Genel Bakış, Bağışlar, Kampanyalar, Kurban Takibi, Ramazan Yardımları, Bağışçılar, Gönüllüler, Dekont Bildirimleri, Raporlar, Galeri / Medya, Mesajlar, Sayfa İçerikleri, Ayarlar, Kullanıcılar, İşlem Kayıtları, Çıkış Yap.
